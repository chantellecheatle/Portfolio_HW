My Skills

User research starts, first and foremost with the user and how they interact with an online platform. Performing user testing and interviews to uncover the core issues with a site is a vital starting point in any design process. Doing so leads to seamless user friendly design, creating a beautiful space for both users and stakeholders.

To gain a deeper understanding of the user, the most vital step in user research is interviews. A series of questions are used to delve deeper into user pain points and frustrations. Gaining this knowledge early in the design process leads to a product which will satisfy your users needs.
