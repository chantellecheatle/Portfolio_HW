Tenant

A Platfrom to bridge the communication gap between tenants and landlords

By Chantelle Cheatle

The aim of this project was to create a simple, easy to use platform to bridge the communication gap between tenant and landlord. A consolidated communication platform tenant first and takes care of their needs. The product was designed to offer rentees a powerful tool to develop and maintain a positive relationship with renters. A team of 3 User Experience and User Interface designers collaborated to create the communication platform, Tenant.
My Roles: User research and interviews Gathering user insights Feature prioritization Wireframing and prototyping Tools Used: Miro, Figma Adobe XD Trello Timeline: 2 weeks

Gathering Information

After gathing user insights we were able to delve depper into what the users wanted from a communication platform. A software which would allow them to have direct contact with their landlord which is reliable, easy to track and would hold both parties accountable.

Ideation and Prototyping

The key component in the ideation process was creating a feature prioritization map, highlighting what the user would benefit from most through the app, whilst also benefiting stakeholders. Having now gathered the key features the prototyping phase was under way.

The Final Product

Through out the prototyping the main aim was to keep the design simple for the user. To ensure the designs were user friendly a lo-fidelity prototype was developed using wireframing. Using the first prototype user tesing was again conducted. Users were set 3 simple tasks to complete through the app to ensure easy accessability. Once we could ensure a simple user friendly design had been achieved we were able to begin pushing the prototype to hi-fidelity to present to stakeholders.
